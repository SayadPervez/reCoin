# reCoin
---
### Requirements:
#### Node JS
####    |--------> Socket.io
####    |--------> Socket.io-Client
####    |---------> Express Module
#### Python
#### Mingw C++ Compiler
---
### Languages:
#### JavaScript for WebPage and Server Design.
#### Python for Emailing
#### C++ for faster access to the Database
> All these languages work together by interacting with underlying the Operating Systems' Command Line
